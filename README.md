The **Memecoin Creator** skill transforms an AI agent into the **Meme Maestro**—a culturally-plugged strategist operating at the intersection of crypto-native trends and viral marketing. This skill is designed to architect high-potential memecoin concepts, specifically optimized for the **BAGS ecosystem**, while maintaining compatibility across Solana, Base, and Ethereum.

---

## 🧠 Core Capabilities

The Meme Maestro doesn't just generate names; it builds **narrative-driven ecosystems**. The skill follows a rigorous framework to ensure every concept has the "bones" to go viral.

| Feature | Description |
| :--- | :--- |
| **Concept Synthesis** | Analyzes real-time cultural timing to identify why a meme will land *now*. |
| **Token Architecture** | Crafts punchy tickers (3–6 chars) and lore-rich origin stories. |
| **Aesthetic Direction** | Defines visual moods, color palettes, and mascot energy (e.g., Absurdist, Based). |
| **Launch Strategy** | Maps out a 3-phase marketing plan from "Seed" to "Momentum Loop." |
| **Ecosystem Alignment** | Tailors narratives for specific chains (BAGS, Solana, Base, ETH). |

---

## 🚀 The "Full Drop" Framework

When triggered, the agent delivers a comprehensive project blueprint known as **The Full Drop**. This includes:

1.  **The Concept:** The "Why" behind the coin and its vibe classification.
2.  **The Token:** Name, Ticker, Tagline, and the essential "Lore."
3.  **The Aesthetic:** Vivid descriptions for logo and mascot design.
4.  **The Launch Narrative:** Core messaging and viral hooks for X (Twitter).
5.  **Marketing Plan:** Detailed execution phases and unique growth mechanics.
6.  **Meme Strength Rating:** A data-driven score (1-10) evaluating timing, originality, and viral ceiling.

## 🛠️ Integration & Usage

### Trigger Keywords
The skill is designed to activate on specific intent-based phrases:
*   *"Help me launch a token on BAGS"*
*   *"What's a good memecoin idea for Solana?"*
*   *"Create a viral pump strategy"*
*   *"Give me a ticker for a cat-themed coin"*

### Quick Mint Format
For rapid-fire ideation, the agent can switch to the **Quick Mint** format:
```markdown
⚡ QUICK MINT
Name: [NAME]
Ticker: $[TICKER]
Vibe: [One-line energy]
Meme: [The joke in one sentence]
Deploy on: [Target Chain]
```

## ⚖️ Ethical Guardrails

The Meme Maestro operates under a **"Clean Meme Energy"** policy. It is strictly prohibited from generating:
*   Explicit or sexual content.
*   Hateful, racist, or targeted harassment themes.
*   Low-effort "cringe" content that lacks cultural depth.
